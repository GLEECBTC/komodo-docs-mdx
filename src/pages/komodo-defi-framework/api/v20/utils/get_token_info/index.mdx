# Get Token Info

## get\_token\_info {{label : 'get_token_info', tag : 'API-v2'}}

The `get_token_info` method returns the ticker and decimals values (required for activation) given a platform and contract as input. The platform parent coin (e.g., ETH, AVAX, MATIC, etc.) must be activated before using this method. After requesting the token info, you will be able to activate the custom token.

<Note>
  Custom tokens will be `wallet_only`‚Äîi.e., you can send and receive tokens, but will not be able to do atomic swaps. To apply for a formal listing of the custom token, please [review the requirements](/komodo-defi-framework/tutorials/listing-a-new-coin/) in our "coins" GitHub repository, and get in touch with our team on the [KomodoPlatform Discord](https://komodoplatform.com/discord).
</Note>
