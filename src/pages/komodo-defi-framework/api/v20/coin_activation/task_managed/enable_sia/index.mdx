# Task: SIA Activation

The task-managed SIA activation flow consists of:

*   `task::enable_sia::init`: Start activation, returns a `task_id`
*   `task::enable_sia::status`: Poll activation progress and final result
*   `task::enable_sia::user_action`: Provide user action when required (reserved for SIA)
*   `task::enable_sia::cancel`: Cancel an ongoing activation

Refer to each subpage for arguments, responses, and examples.
