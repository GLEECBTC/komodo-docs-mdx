# task::enable\_sia::user\_action {{label : 'task::enable_sia::user_action', tag : 'API-v2'}}

If the `task::enable_sia::status` returns `UserActionRequired`, use this method to provide
the required user action payload.

Note: Hardware-wallet user interactions are not currently required for SIA activation.
